Proyek Saya

Aplikasi Ticketing Keluhan

Sebuah aplikasi ticketing untuk posting keluhan dari user ke admin. Sebuah aplikasi web dinamis yang di-hosting dan diuji menggunakan server Apache di XAMPP. Front-end-nya menggunakan HTML/CSS/JS untuk berinteraksi dengan API, sementara MySQL (via XAMPP) mengelola seluruh database-nya.

Aplikasi Kasir

Sebuah aplikasi kasir sederhana untuk toko. Aplikasi ini ditenagai oleh micro-framework Flask, yang menangani routing dan live server. Untuk manajemen data, saya memanfaatkan SQLAlchemy (ORM Python) untuk membuat dan berinteraksi dengan database SQLite secara terprogram.
